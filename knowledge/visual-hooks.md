# 시각 훅 패턴 리스트

화면 자체로 스크롤을 멈추게 하는 5가지 방법.
텍스트 훅이 "어떤 말을 하느냐"로 잡는 거라면, 시각 훅은 "어떤 화면을 보여주느냐"로 시청자를 사로잡습니다.

---

## 1. 카메라 앵글
평소에 잘 못 보던 앵글이 나오는 순간 눈이 멈춤.
우리 뇌는 익숙한 건 무시하고, 평소와 다른 화면에 자동으로 주의를 기울임.

| 앵글 | 느낌 | 적합 카테고리 |
|---|---|---|
| 하이앵글 (위에서 아래) | 아기자기, 귀여움 | 음식, 루틴, 데스크 |
| 로우앵글 (아래에서 위) | 강인함, 역동적 | 피트니스, 스포츠 |
| 클로즈업 | 디테일 강조, 감정 전달 | 뷰티, 감성 |

같은 장면이라도 앵글 하나 바꾸는 것만으로 완전히 다른 영상처럼 보일 수 있음.

**테리크 적용:**
- 하이앵글: 수건 세탁 과정, 포장 언박싱, 제품 구성 전체
- 클로즈업: 수건 소재 질감, 색상 디테일, 자수
- 정면 토킹헤드: 정보 설명, Q&A

---

## 2. 높은 채도의 색감
채도가 높은 화면은 일단 그냥 눈에 들어옴.
칙칙하고 어두운 화면은 스크롤하면서 그냥 지나치게 되지만, 선명하고 강렬한 색감은 무의식적으로 시선을 끔.

| 상황 | 채도 방향 |
|---|---|
| 타겟이 어리거나 야식/디저트/게임 카테고리 | 채도 높이기 → 즉각적 도파민 자극 |
| 프리미엄 타겟 / 신뢰·전문성 중요 | 채도보다 톤의 일관성이 더 중요 |

**테리크 적용:** 30~50대 여성 타겟 + "Colorful Classic" 브랜드 → 톤 일관성 유지하면서 테리크 수건의 컬러풀한 색감 강조. 자연광에서 화이트/베이지 배경으로 색감 극대화.

---

## 3. 동적 후킹
인간은 본능적으로 멈춰있는 것보다 움직이는 것에 시선이 끌리도록 설계되어 있음.

| 방법 | 설명 |
|---|---|
| 빠른 컷 전환 | 시작과 동시에 화면이 바뀌면 눈이 잡힘 |
| 화면 안에서 뭔가 일어나는 장면 | 사건/상황 자체가 동적 자극 |
| 작은 애니메이션 | 토킹헤드 영상도 작은 동적 자극으로 몰입감 부여 |

**테리크 적용:**
- 여러 색상 수건을 빠르게 보여주는 컷 전환
- 수건 펼치는 장면 → 펄럭이는 느낌
- 섬유유연제 vs 식초 붓는 비교 장면

---

## 4. 비주얼 매력
몸매, 예쁜 얼굴, 감각적인 인테리어, 맛있어 보이는 음식 등 비주얼 자체가 훅이 되는 경우.

| 항목 | 설명 |
|---|---|
| 적합 카테고리 | 피트니스, 뷰티, 패션, 인테리어, 요리, 여행 |
| 핵심 | 첫 장면부터 매력적인 비주얼을 바로 보여줘야 함 |
| 표정의 중요성 | 비슷한 영상이라도 웃는 얼굴 vs 무표정의 조회수 차이가 엄청남 |

촬영할 때 의식적으로 웃는 것만으로도 조회수가 달라질 수 있음.

**테리크 적용:**
- 깔끔하게 정렬된 컬러풀한 수건 더미 첫 장면
- 보드라운 질감을 손으로 만지는 클로즈업
- 욕실에 예쁘게 걸린 수건 세팅 감성 장면

---

## 5. 흥미로운 장면 및 상황
뭔가 일어나고 있는 장면, 예상 밖의 상황. 보자마자 '이게 뭐야? 어떻게 된 거지?' 생각이 드는 장면.

| 항목 | 설명 |
|---|---|
| 적합 | 브이로그, 일상, 바이럴 |
| 핵심 | 결말이 궁금해서 멈추게 됨 |
| 팁 | 하이라이트를 처음에 배치하는 게 핵심 |

**테리크 적용:**
- 뻣뻣한 수건 vs 테리크 수건 촉감 비교 실험 장면
- 답례품 박스 개봉 장면
- "섬유유연제 넣었더니 이렇게 됐어요" 결과 장면을 먼저 보여주기

---

## 촬영 구도 조합 레퍼런스

### 수건 정보 콘텐츠 (교육팁/통념파괴)
→ 정면 토킹헤드 (훅+CTA) + 클로즈업 (수건 디테일) + 하이앵글 (세탁 과정)

### 수건 제품 소개형
→ 와이드 (제품 전체 구성) + 클로즈업 (소재, 자수) + 정면 토킹헤드 (설명)

### 답례품 커머스 스토리텔링
→ 정면 토킹헤드 (썰 시작) + 하이앵글 (제품 포장/구성) + 클로즈업 (디테일)

### 답례품 언박싱/리뷰
→ 하이앵글 (박스 개봉) + 클로즈업 (제품 디테일) + 정면 (마무리 멘트)

---

## 테리크 브랜드 촬영 팁
- 배경: 흰 벽 또는 자연광 욕실, 깔끔한 욕실 선반
- 색감: 자연광에서 테리크 수건의 컬러풀한 색감 강조
- 질감: 클로즈업으로 수건 표면의 도톰함과 부드러움 표현
- 브랜드감: 여러 색상 수건을 함께 배치해 "Colorful Classic" 감성 연출
- 표정: 항상 웃는 얼굴로 시작 (조회수에 직접 영향)
