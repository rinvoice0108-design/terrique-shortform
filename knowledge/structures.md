# 영상 구조 템플릿 15종

터진 영상의 대본 구조를 분해한 템플릿입니다.
스토리텔링 6종은 퍼스널 브랜딩에, 단계별 9종은 정보성/세일즈 영상에 적합합니다.

---

## 원고 핵심 원칙
- 훅(Hook): 첫 3초 안에 스크롤을 멈춰야 함
- 5요소: 훅 → 근거 → 시청이유 → 가치증명 → CTA
- 5요소를 대본에 전부 넣을 필요 없음 (캡션/제목텍스트에 분산 배치 가능)
- 구어체로 작성 (소리 내어 읽어서 자연스러운 것만)
- AI스러운 문장 금지

## 📖 스토리텔링 구조 (6종)

### 1. 성장 서사 구조
"나도 그랬다 → 이렇게 해결했다 → 지금은 이렇다"

| 단계 | 설명 | 템플릿 |
|---|---|---|
| 1. 훅 | 과거의 나 or 충격적 실패로 시작 | 나는 [과거 상태/충격적 실패]였다 |
| 2. 고통 | 구체적 시련과 실패한 시도들 | [시도 1, 2]를 해봤지만 전부 실패했다 |
| 3. 전환점 | 깨달음 or 핵심 원리 발견 | 마침내 [핵심 깨달음/방법]을 발견했다 |
| 4. 변화 | Before → After 구체적 성과 | [이전]에서 [이후]로 변했고 [성과]를 얻었다 |
| 5. 교훈/CTA | 배운 점 공유 or 비즈니스 연결 | [교훈]을 얻었다 / 같은 분들을 돕기 위해 [비즈니스]를 시작했다 |

변주: A. 성장 서사(4번에 무게) / B. 역경 극복(1~2번에 무게) / C. 교훈/배움(5번에 무게)

---

### 2. 자기소개 구조
내가 왜 이 일을 하는지, 나는 어떤 사람인지

| 단계 | 설명 | 템플릿 |
|---|---|---|
| 1. 도입 | 평범했던 과거 | X년 전 나는 [직업/상황]이었다 |
| 2. 갈등 | 변화의 계기가 된 사건 | [생각]이 들어 [행동]을 하였다 |
| 3. 깨달음 | 관점의 변화 | 그 일련의 사건으로 나는 [깨달음]을 얻었다 |
| 4. 변화 | 깨달음 후 취한 행동과 현재 | 그래서 [행동]을 했고, 지금은 [결과]를 얻었다 |
| 5. 목적 | 브랜드 사명 선포 | 나의 지식이 필요한 사람들에게 [도움]이 되고자 한다 |

### 3. 거대한 꿈/챌린지 구조
목표를 선포하고 과정을 보여주는 구조

| 단계 | 설명 | 템플릿 |
|---|---|---|
| 1. 꿈 소개 | 꿈의 기원 | 나는 지금부터 [꿈]을 할것이다 |
| 2. 도전 | 행동 시작 계기 | 어느 날 결심하고 [행동]을 시작했다 |
| 3. 진행 상황 | 현재 성과와 남은 목표 | 지금까지 [성과]를 냈고, 앞으로 [남은 단계]를 해야 한다 |
| 4. CTA | 응원과 참여 유도 | 내가 [꿈]을 할 수 있을까? / CTA |

### 4. 획기적 발견 구조
반복된 실패 끝에 해결책을 발견하는 구조

| 단계 | 설명 | 템플릿 |
|---|---|---|
| 1. 문제 | 반복되던 실패 상황 | 계속 반복되는 상황, [나쁜 결과]였다 |
| 2. 발견 | 놀라운 깨달음/결과 | [N일/개월]만에 만든 [결과]이다 |
| 3. 해결 | 구체적 방법과 결과 | 구체적인 [해결책]을 제시, ~이렇게 해보세요 |

### 5. 커머스 스토리텔링 구조 ★ [답례품 1순위]
경험담처럼 시작해서 자연스럽게 제품/서비스로 연결하는 구조. 광고 느낌 없이 구매 전환을 유도하는 핵심 구조.

| 단계 | 설명 | 템플릿 |
|---|---|---|
| 1. 썰 훅 | 경험/사건처럼 시작 (광고 느낌 X) | "얼마 전에 [경험/사건]이 있었는데..." |
| 2. 공감 | 시청자도 겪는 문제로 확장 | "다들 [문제/고민] 있잖아요" |
| 3. 발견 | 자연스럽게 해결책 등장 | "근데 [제품/방법]을 써보니까..." |
| 4. 소구점 | 핵심 장점 1~2개만 | "[구체적 변화/효과]가 진짜 달라졌어요" |
| 5. 증명 | B&A or 사용 장면 | 실제 사용 모습 / 변화 과정 |
| 6. 부드러운 CTA | 강매 아닌 정보 제공 | "궁금하신 분 댓글 남겨주세요" / 프로필 링크 |

핵심: 처음~중간까지 광고인 줄 모르게 진행. "판매"가 아니라 "경험 공유" 느낌이 포인트.
테리크 예시: 결혼 답례품 준비, 돌잔치 답례품 수건 추천

---

### 6. 프로세스/서비스 소개 구조
"저는 이런 일을 합니다"를 과정 전체를 보여주며 소개하는 구조.

| 단계 | 설명 | 템플릿 |
|---|---|---|
| 1. 소개 훅 | 직업/서비스를 한 줄로 | "저는 [직업]입니다. [서비스]가 어떻게 진행되는지 보여드릴게요" |
| 2. STEP 1 | 첫 번째 과정 | [상담/문의/진단] 단계 |
| 3. STEP 2~N | 핵심 과정 빠르게 나열 | [기획/제작/시술/관리] 과정을 빠른 컷으로 |
| 4. 결과물 | 완성된 모습/결과 | 최종 [결과물/변화] 공개 |
| 5. CTA | 문의/예약 유도 | "궁금하신 분은 프로필 링크에서 문의하세요" |

핵심: 과정 자체가 콘텐츠. 빠른 컷 전환으로 몰입감 있게 보여주는 게 포인트.

### 7. 교육팁/노하우 구조 ★ [수건 정보 2순위]
실질적인 정보를 단계별로 전달하는 구조

|---|---|
| 1. 훅 | 정보 소개 |
| 2. 팁 | 무엇을 가르쳐줄지 요약 |

테리크 예시: 수건 빨리 말리는 법, 호텔 수건처럼 관리하는 법

### 8. 통념파괴 구조 ★ [수건 정보 1순위]
잘못된 상식을 깨고 올바른 해결책을 제시하는 구조

| 단계 | 설명 |
|---|---|
| 1. 훅 | 강렬한 문구로 시작 ("아직도 이렇게 하세요?" / "이거 하면 안 됩니다") |
| 2. 오해/실수 | 사람들이 믿는 잘못된 정보 or 흔한 실수 언급 |
| 3. 반박 | 왜 틀렸는지 근거와 함께 설명 |
| 4. 해결책 | 대신 무엇을 해야 하는지 올바른 방법 제시 |
| 5. CTA | 내 서비스/채널로 연결 or 팔로우 유도 |

테리크 예시: 섬유유연제가 수건을 망친다, 수건 매일 세탁 안 해도 됩니다

### 12. 상품/서비스 세일즈 구조 ★ [답례품 2순위]
고민을 짚고 내 상품이 해결책임을 어필하는 구조

| 단계 | 설명 |
|---|---|
| 1. 훅 | 핵심 불편/고민 언급 |
| 2. 해결책 | 해결 방법이 있음을 알림 |
| 3. 가치 | 실행 가능한 단계별 솔루션 제시 |
| 4. 홍보 | 내 상품/서비스가 그 해결책임을 어필 |
| 5. CTA | 구매/문의 유도 |

테리크 예시: 답례품 고민이라면, 답례품 수건 고르는 기준

### 13. Q&A / 댓글 답변 구조 ★ [수건·답례품 모두 가능]
팔로워의 질문에 전문가로서 답변하는 구조. 소통감 + 권위를 동시에 잡음.

| 단계 | 설명 |
|---|---|
| 1. 훅 | "이 질문이 가장 많았어요" / "댓글에서 [질문]이 왔는데" |
| 2. 질문 소개 | 실제 댓글/DM 캡처 or 질문 텍스트 노출 |
| 3. 답변 | 전문가적 관점에서 명쾌한 답변 |
| 4. 추가 팁 | "덧붙이자면..." 보너스 정보 |
| 5. CTA | "궁금한 거 더 물어봐" 댓글 유도 |

핵심: 실제 댓글/DM을 보여주면 소통하는 채널이라는 인식 + 콘텐츠 소재 무한 생성.

### 14. 하루 루틴/타임라인 구조
시간순으로 일상을 보여주며 공감과 친밀감을 만드는 구조.

| 단계 | 설명 |
|---|---|
| 1. 훅 | "[나이] [직업]의 하루" / "[직업]의 출근 전 1시간" |
| 2. 타임라인 | 시간순 나열 (06:00 → 07:00 → ...) |
| 3. 하이라이트 + 마무리 | 하루 중 가장 인상적인 순간을 길게 잡아서 자연스럽게 마무리 |

핵심: 시간 표시가 들어가서 구조감 있음. 편집 템포가 빨라야 지루하지 않음.

### 15. 리뷰/언박싱 구조
제품이나 서비스를 솔직하게 평가하는 구조.

| 단계 | 설명 |
|---|---|
| 1. 훅 | "드디어 도착한 [제품]" / "[제품] 솔직 리뷰" |
| 2. 첫인상 | 패키지/외관/첫 느낌 |
| 3. 사용 | 실제 사용 과정과 체감 |
| 4. 장단점 | 솔직한 평가 (장점만 나열하면 신뢰 X) |
| 5. 결론 | 추천 여부 + 점수 or 한 줄 평 |

핵심: 장점만 말하면 광고처럼 보임. 단점도 솔직하게 말해야 신뢰도가 올라감.
테리크 예시: 테리크 수건 솔직 언박싱, 답례품 개봉 리뷰
