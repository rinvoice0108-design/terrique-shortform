# 촬영 구도 가이드

같은 내용이라도 구도 하나로 느낌이 완전히 달라집니다.
숏폼에서 자주 쓰이는 5가지 촬영 구도와 각 구도별 적합한 상황을 정리했습니다.

---

## 구도 5가지

### 1. 정면 토킹헤드 (Eye-Level)
카메라와 눈높이를 맞추고 정면을 보며 말하는 가장 기본적인 구도.

| 항목 | 설명 |
|---|---|
| 느낌 | 신뢰감, 친근함, 대화하는 느낌 |
| 적합한 콘텐츠 | 정보 전달, 교육, Q&A, 자기소개, 리뷰 |
| 촬영 팁 | 렌즈를 눈으로 보고 말하기. 눈 아래로 내려다보는 각도 X. 상반신 프레이밍이 기본 |
| 주의점 | 배경이 지저분하면 신뢰도 하락. 깔끔한 배경 or 블러 필수 |

---

### 2. 하이앵글 (위에서 아래로)
카메라를 위에서 아래로 내려다보는 구도. 탁자 위 작업이나 제품을 보여줄 때 최적.

| 항목 | 설명 |
|---|---|
| 느낌 | 아기자기, 정돈된, 감성적 |
| 적합한 콘텐츠 | 요리, 언박싱, 데스크 셋업, 뷰티(손 작업), 플랫레이 |
| 촬영 팁 | 삼각대 + 폰 거치대 필수. 손 동작이 들어가면 몰입감 상승. 그림자 주의 |
| 주의점 | 얼굴이 나오는 영상에는 부적합 (이마가 넓어 보임) |

**테리크 적용:** 수건 세탁 과정, 포장 언박싱, 제품 구성 전체, 식초 붓는 장면

---

### 3. 로우앵글 (아래에서 위로)
카메라를 낮은 위치에서 위로 올려다보는 구도. 피사체를 크고 강하게 보이게 함.

| 항목 | 설명 |
|---|---|
| 느낌 | 강인함, 역동적, 권위, 웅장함 |
| 적합한 콘텐츠 | 피트니스, 스포츠, 패션(전신 핏), 건물/공간 소개 |
| 촬영 팁 | 무릎~허리 높이에서 촬영. 운동 동작이나 전신 코디 영상에 효과적 |
| 주의점 | 너무 낮으면 부자연스러움. 턱선이 강조되어 얼굴이 커보일 수 있음 |

---

### 4. 클로즈업 (초근접 디테일)
피사체의 특정 부분을 극도로 가깝게 촬영하는 구도. 디테일과 감정을 극대화.

| 항목 | 설명 |
|---|---|
| 느낌 | 몰입감, 전문성, ASMR 감성, 섬세함 |
| 적합한 콘텐츠 | 뷰티(메이크업/스킨케어), 요리(재료 컷), 공예/핸드메이드, 시술 과정, 제품 디테일 |
| 촬영 팁 | 손 떨림 방지 필수 (삼각대 or 안정화). 조명이 핵심 — 그림자가 디테일을 죽임 |
| 주의점 | 클로즈업만 연속되면 답답함. 다른 구도와 교차 편집 필수 |

**테리크 적용:** 수건 소재 질감, 색상 디테일, 자수, 수건 표면의 도톰함

---

### 5. 와이드/풀샷 (전체 공간)
넓은 공간이나 전체 상황을 한 프레임에 담는 구도.

| 항목 | 설명 |
|---|---|
| 느낌 | 시원함, 개방감, 상황 설명, 여행 감성 |
| 적합한 콘텐츠 | 여행, 공간 투어(매장/카페/집), 브이로그 전환 장면, 운동 전신 동작 |
| 촬영 팁 | 광각 렌즈 활용 시 더 임팩트. 인물이 들어가면 스케일감 부여. 이동 샷과 궁합 좋음 |
| 주의점 | 피사체가 너무 작아지면 집중력 분산. 구도 안에 시선을 끌 포인트가 있어야 함 |

**테리크 적용:** 욕실 세팅 전체, 제품 구성 전체, 수건 여러 색상 한번에 보여주기

---

## 구도 조합 가이드

하나의 영상에 구도 1개만 쓰는 것보다, 2~3개씩을 조합하면 훨씬 역동적입니다.

| 영상 유형 | 추천 구도 조합 |
|---|---|
| 정보 전달 / 교육 | 정면 토킹헤드 + 클로즈업 (자료 화면) |
| 제품 리뷰 | 정면 토킹헤드 + 하이앵글 (제품) + 클로즈업 (디테일) |
| 요리 | 하이앵글 (조리 과정) + 클로즈업 (재료/완성) |
| 운동 / 피트니스 | 로우앵글 (동작) + 와이드 (전신) + 정면 (설명) |
| 브이로그 / 여행 | 와이드 (장소) + 정면 (이야기) + 클로즈업 (음식/디테일) |
| 서비스 프로세스 | 와이드 (공간) + 클로즈업 (시술/작업) + 정면 (설명) |

---

## 테리크 브랜드 추천 구도 조합

| 콘텐츠 유형 | 추천 구도 조합 |
|---|---|
| 수건 정보 (통념파괴/교육팁) | 정면 토킹헤드 (훅+CTA) + 클로즈업 (수건 디테일) + 하이앵글 (세탁 과정) |
| 수건 제품 소개 | 와이드 (제품 전체 구성) + 클로즈업 (소재, 자수) + 정면 토킹헤드 (설명) |
| 답례품 커머스 스토리텔링 | 정면 토킹헤드 (썰 시작) + 하이앵글 (제품 포장/구성) + 클로즈업 (디테일) |
| 답례품 언박싱/리뷰 | 하이앵글 (박스 개봉) + 클로즈업 (제품 디테일) + 정면 (마무리 멘트) |
| Q&A | 정면 토킹헤드 (전체) + 클로즈업 (관련 제품/자료) |
